# Automation Framework Installer

## How to Run
```
npm install
npm start
```
![Screenshot 2025-04-10 112939](https://github.com/user-attachments/assets/2734df0c-750d-4467-a5fd-1f60467bd49f)

## Choose Desirable Framework
```
Cypress (UI)
Playwright (UI)
Selenium (UI)
Mocha (API)
Cucumber (API)
Godog (API)
WebdriverIO (Apps)
Pytest (API)
Behave (API & UI)
```
